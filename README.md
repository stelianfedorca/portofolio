# Personal Portfolio

A modern, responsive portfolio website built with Next.js 16, featuring smooth animations, dark/light theme support, and a clean, professional design.

## Features

- **Modern Tech Stack**: Built with Next.js 16, React 19, and TypeScript
- **Responsive Design**: Fully responsive layout that works seamlessly on all devices
- **Theme Toggle**: Dark/light mode with smooth transitions and persistent preferences
- **Smooth Animations**: Powered by Framer Motion for elegant page transitions and interactions
- **Styled Components**: CSS-in-JS with styled-components for modular, maintainable styling
- **Mobile-First Navigation**: Sleek dropdown menu with click-outside detection
- **Custom Hooks**: Reusable React hooks for enhanced functionality
- **Glassmorphism UI**: Modern design with backdrop blur effects and floating elements
- **Optimized Performance**: Built with Next.js for fast loading and SEO optimization

## Built With

- **[Next.js](https://nextjs.org/)** - React framework for production
- **[React](https://react.dev/)** - UI library
- **[TypeScript](https://www.typescriptlang.org/)** - Type safety
- **[Styled Components](https://styled-components.com/)** - CSS-in-JS styling
- **[Framer Motion](https://www.framer.com/motion/)** - Animation library
- **[React Icons](https://react-icons.github.io/react-icons/)** - Icon library
